# Team1 Mini-Grant Interview Playbook (Devil_World)

## 5-7 Minute Pitch Deck Flow
### Slide 1: Vision
"Devil_World is a playable action-adventure where progression is both skill-based and verifiably on-chain."

### Slide 2: Problem
- Web3 games often lack gameplay depth
- Traditional games lack interoperable ownership

### Slide 3: Solution
- Action gameplay systems + live blockchain actions
- AVAX staking, NFT ownership, and CIT token economy integrated into progression

### Slide 4: Why Avalanche
- Fuji C-Chain for rapid, low-cost iteration
- EVM tooling compatibility for fast delivery
- Path to Avalanche L1 when custom economics/performance become mandatory

### Slide 5: Traction / Build Status
- Functional prototype with wallet, contracts, and game loop integrated
- Demo-ready transaction flows and achievement UX

### Slide 6: Milestones
- Near-term: polish, telemetry, retention loops
- Mid-term: seasonal events and token sinks
- Long-term: mainnet launch + optional L1 strategy

## Technical Architecture Walkthrough (8-10 min)
1. Frontend game loop and systems
2. Wallet and network enforcement (Fuji only)
3. Contract service abstraction and tx lifecycle
4. Smart contract responsibilities and security controls
5. Gameplay-to-blockchain data flow

## Live Demo Runbook (Safe + Repeatable)
1. Connect MetaMask (Fuji)
2. Start run and show combat / movement / world systems
3. Press `1` stake AVAX
4. Press `3` mint NFT
5. Press `6` mint CIT
6. Press `7` stake CIT
7. Press `2` claim AVAX rewards or `8` claim CIT rewards
8. Press `4` show account balance / proof of state

## Judge Questions You Should Be Ready For
- Why is blockchain necessary in this game?
- Which actions are fully on-chain vs off-chain?
- How do you prevent economy inflation?
- What is your migration path from Fuji to production?
- What are your 30/60/90-day milestones if selected?

## Strong Closing Statement
"We are not adding blockchain as a badge. In Devil_World, blockchain is tied to measurable player progression, ownership, and reward logic, and Avalanche gives us the performance and developer velocity to ship this responsibly."
