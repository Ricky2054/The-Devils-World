## SLIDE 2 — THE PROBLEM (45 seconds)

### Web3 Gaming Is Broken in Two Ways

| Problem | Result |
|---------|--------|
| **Blockchain-first games** lack gameplay depth | Players leave after first mint |
| **Gameplay-first games** bolt on tokens as afterthought | Blockchain adds no real value |
| **Leaderboards are off-chain** | No verifiable competition, no cross-device persistence |
| **Reward economies are opaque** | Players can't verify or own their progress |

**The gap:** No one is building games where blockchain is *structurally necessary* for the core loop — not just cosmetic.

## SLIDE 3 — THE SOLUTION (60 seconds)

### Devil World: Gameplay + On-Chain Ownership in One Loop

```
┌─────────────────────────────────────────────────┐
│  PLAY → KILL → COLLECT → LEVEL UP → MINT NFT   │
│    ↓                                      ↓     │
│  Earn Gold/XP                   Stats go ON-CHAIN│
│    ↓                                      ↓     │
│  STAKE AVAX ←── Rewards ←── LEADERBOARD (global)│
│    ↓                                      ↓     │
│  CIT Tokens ←── Game Economy ←── NFT Ownership  │
└─────────────────────────────────────────────────┘
```

**What makes it different:**
- **Daily Streak System** — login streaks boost gold multiplier (1.1× → 5×)
- **Boss Enemies every 5 levels** — scaled HP, drops gold, 30% NFT drop chance
- **Gold Decay Timer** — 8-second loot windows create urgency
- **On-Chain Leaderboard** — scores stored on Avalanche, visible globally across all devices
- **Achievement NFTs** — mint your stats permanently on blockchain when you're proud of a run

## SLIDE 4 — WHY AVALANCHE (60 seconds)

### Avalanche Is Not Incidental — It's Structural

| Requirement | Why Avalanche Solves It |
|-------------|------------------------|
| **Frequent game transactions** (stake, mint, claim) | Sub-second C-Chain finality — no waiting |
| **Low-cost iteration** | Fuji testnet faucets allow unlimited playtesting |
| **EVM compatibility** | Solidity + ethers.js = fast development velocity |
| **Verifiable competition** | Snowtrace explorer proves every transaction |
| **Scale path** | C-Chain → Avalanche L1 when throughput demands it |

### "What breaks if we remove the blockchain?"
- Leaderboard becomes local-only — no cross-device, no verifiable rankings
- Players cannot own their achievements permanently
- Staking rewards cease — no DeFi integration with gameplay
- Gold economy has no auditable supply mechanics
- NFT-gated features (boss loot drops) stop working
